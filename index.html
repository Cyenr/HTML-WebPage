O que é HTML?

O HTML (Hyper Text Markup Language) não é uma linguagem de programação, mas sim um arquivo de texto que contém delimitadores por nomes que um navegador consegue interpretar. Cada um desses delimitadores têm uma função e comportamento distinto um do outro.

(Voltar)

Coisas que podemos fazer com HTML:

  • Criar estrutura de página: O HTML é usado para criar a estrutura básica de uma página web, incluindo cabeçalhos, parágrafos, listas, tabelas e divs
  • Inserir texto e mídia: Podemos adicionar texto, imagens, vídeos e áudios em páginas web utilizando tags HTML específicas.
  • Links e navegação: O HTML permite a criação de links para outras páginas web, recursos dentro da mesma página ou mesmo para download de arquivos.

Resumidamente falando, o HTML é a base para a criação de páginas web e oferece uma ampla gama de recursos para criarmos experiências online ricas e interativas.

(Voltar)

Aqui está um exemplo de estrutura base de um HTML:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
</head>
<body>
</body>
<html>

Declaração de documento

  • Todo documento HTML DEVE começar com <!DOCTYPE html>, pois é ela que informará o navegador sobre o tipo de documento que ele irá ler;
  • <!DOCTYPE html> não é uma tag, mas sim uma declaração de documento;
  • <!DOCTYPE html> NÃO é "case sensitive".

<html>

  • A tag <html> representa a raíz de um doumento HTML;
  • É o contêiner para todos os outros elementos* HTML (exceto a declaração <!DOCTYPE html>);
  • Devemos sempre incluir o atributo lang* dentro da tag <html> para declarar a língua da página web.

<head>

  • A tag <head> é um contêiner para metadados e está localizado entre a tag <html> e a tag <body>;
  • Metadados são dados sobre o documento HTML, que não são exibidos ao usuário;
  • Existem algumas tags que podem estar dentro da <head>, como <title>, <style>, <script>, etc..
  • <title> é uma tag obrigatória em todo documento HTML.

<title>

  • Define um título para a barra de ferramentas do navegador;
  • Define um título para a página quando é adicionada aos favoritos
  • Exibe um título para a página nos resultados do mecanismo de pesquisa

<body>

  • Define o corpo do documento HTML;
  • A tag <body> contém todo o conteúdo de um documento HTML, como parágrafos, imagens, hiperlinks, tabelas, listas, etc.
  • Só pode haver um <body> em um documento HTML.

Essa é a estrutura base de um documento HTML, com isso, já podemos criar uma página web.

Mais Tags HTML

Existem algumas outras tags HTML que podem ajudar de acordo com o objetivo da página que estamos criando. A seguir estarei listando mais algumas tags:

<h1> ao <h6>

  • São usados para definir o tamanho de um título ou subtítulo;
  • <h1> define o título principal enquanto o <h2> em diante se tornam subtítulos;
  • Uma boa prática é haver apenas um único <h1>.

<p>

  • Define um parágrafo no documento HTML;
  • O navegador define uma única linha em branco para cada <p>;
  • Caso necessário haver quebra de linha, deve-se usar uma outra tag (<br>).

<br>

  • Adiciona uma única quebra de linha em um parágrafo;
  • Diferente da maioria das tags, a <br> é uma tag vazia, onde não existe um fechamento, como <h1> </h1>
  • Use-a para quebra de linha e não para espaçamento entre parágrafos.

<hr>

  • Adiciona uma linha horizontal no documento HTML;
  • É usada para separação de conteúdo na página;
  • Também é uma tag vazia.

<blockquote>

  • Especifíca uma seção para citar as suas fontes;
  • Navegadores geralmente mentêm as linhas de <blockquote> identadas.

Tags de formatação de texto

<i>

  • Deixa o texto, ou parte dele, em itálico;
  • Frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, etc.

<u>

  • Deixa o texto, ou parte dele, sublinhado;
  • É recomendado não usar em casos onde há hiperlinks, pois pode confundir.

<mark>

  • Define o texto, ou parte dele, que deve ser marcado ou destacado.

<small>

  • Define o texto, ou parte dele, como um texto menor;
  • Costuma ser usado para comentários secundários ou, até mesmo, descrever um direito autoral, por exemplo.

<strong>

  • É usado para dar um destaque importante para um texto ou palavra;
  • Se quisermos deixar em negrito, devemos usar uma outra tag (<b>)

<sup>

  • Define um texto sobrescrito;
  • O texto sobrescrito aparece meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor;
  • Texto sobrescrito pode ser usado para notas de rodapé.

<sub>

  • Define um texto subscrito;
  • O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor;
  • O texto subscrito pode ser usado para fórmulas químicas, como H2O.
(Voltar)

Criando listas

<ul>

  • Tag usada para criar uma lista não ordenada;
  • Você pode adicionar itens à sua lista usando <li>.

<ol>

  • Tag usada para criar uma lista ordenada;
  • Como nas listas não ordenadas, pode-se usar <li> para adicionar itens à lista.

<li>

  • Define uma lista de itens;
  • É usada dentro das listas (<ul>) e (<ol>);
  • Em (<ul>) a lista irá ser exibida em tópicos;
  • Em (<ol>) a lista irá ser exibida em números ou letras.

Aqui está um extra!

Existem algumas tags que podemos utilizar no nosso documento HTML como uma forma de interação com o usuário, criando caixas de textos, imagens e links, tanto para outras páginas quanto para uma possível outra página do seu projeto!

<input>

  • Cria um campo de entrada onde o usuário pode inserir dados;
  • É um dos mais importantes elementos* do HTML;
  • Pode ser exibido de diversas maneiras, dependendo do atributo type*;

<img>

  • É usado para adicionar uma imagem no documento;
  • A tag tem dois atributos importantes: src e alt;
  • src especifica o caminho (URL) para a imagem;
  • alt especifica um texto alternativo para a imagem, caso, por algum motivo, ela não puder ser exibida
  • É importante especificar dois atributos (width e height)*;

<a>

  • É a tag responsável em definir hiperlinks, onde são usados para navegar de uma página a outra;
  • O atributo mais importante para a tag é o href, que indica o destino do link;
  • Outro atributo útil da tag <a> é o target
  • Dentro do target existem dois importantes valores (_blank e _self);
  • "_blank" abre o link em uma nova página;
  • "_self" abre o link na mesma página;

Menções

elementos: apenas mais um nome usado para nos referirmos às tags HTML.

lang: linguagem de interpretação do navegador.

  • <html lang="pt-BR"> (português do Brasil)
  • <html lang="en"> (inglês)
  • <html lang="fr"> (francês)

type: é o atributo que especifica o tipo do <input>. Se caso não houver um atributo específico, o tipo padrão será o type="text"

  • type="button" (cria um botão clicável)
  • type="checkbox" (cria uma caixa de seleção)
  • type="password" (cria um campo para inserção de senha)

width: é o atributo responsável no dimensionamento da altura da imagem.

height: é o atributo responsável no dimensionamento da largura da imagem.
